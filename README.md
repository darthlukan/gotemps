# GoTemps

> Author: Brian Tomlinson <brian.tomlinson@linux.com>

## Description

> GoTemps is a command line temperature converter.  It converts between Fahrenheit and Celsius given
input like 32C, 0c, 70F, 50f.

## Usage

    $ gotemps 5
    >> Converted 5C to 41F.
